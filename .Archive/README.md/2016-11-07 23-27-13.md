# 仿酷狗UI效果


## 运行环境
- [Cocoapods](https://github.com/CocoaPods/CocoaPods) 1.0.1
- iOS 10 / Mac OS X 10.12
- Xcode 8.1

##运行
在终端里面运行 `pod install`，然后运行`ZMusic.xcworkspace` 

## 特色
- 左右抽屉效果，共用一个实例。
- 播放页手势左右滑动旋转退出。
- 皮肤包的管理，可网络下载，本地管理，本地更新皮肤包。
- 换肤功能，本地记录最新更换的皮肤。
- 自定义Navigation实现全屏滑动手势的转场动画效果。
- 皮肤预览效果。
